## Description  
  The project is splited in the following modules:  
    - appendFileService  
    
    
## AppendFileService  
### Description  
Mini-application to simulate growing logs, and log rotation  
The target is to serve some log files to a flume agent  
    
### Use  
1. Move into appendFileService directory  
2. Run  `./appendToFileService.init start`
3. If everithing goes right in spoolDir directory file growingFile.txt should be created and will start rotating
